# RinneTenseiCraft
MOBが輪廻転生する。

/rinnetensei on/off\
onで起動 offで停止

ボスモブはバランス調整のために体力100に弱体化\
モブを倒したときに5%の確率でイリュージョナーに変化する。\
ヴェックスは牛/羊/ブタ/ニワトリのそれぞれに25%の確率で転生する。
